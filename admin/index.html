导入数据

下载模板后按 JSON 结构填写，再回到这里预览和导入。导入只处理 Payload、工具命令和导航数据。

1. 下载模板

模板包含可直接填写的完整示例、字段说明和常见错误提醒。JSON 里只有 payloads、tools、navigation、toolNavigation 会真正导入。

导入范围 站点标题、Logo、GitHub/Xeye 等平台信息不会通过导入修改。
填写规则
  • payloads、tools、navigation、toolNavigation 都必须是数组，可一次放很多条。
  • Payload 至少填写 id、name、description、category、execution。
  • 工具至少填写 id、name、description、category、commands。
  • 导航节点用 payloadId 或 toolId 关联对应数据的 id，不写导航也能导入数据，但左侧树不会出现入口。
  • 命令里可以写 {URL}、{COOKIE}、{ATTACKER_IP} 等全局变量，前台展示和复制会自动替换。

2. 选择 JSON 文件

先预览再导入。预览会显示将影响的模块、条目数量和格式提醒。

3. 选择导入方式

合并适合日常追加和更新；覆盖只会覆盖导入文件里包含的模块。

4. 预览与执行

确认预览结果无误后再导入。

选择 JSON 文件后，点击“预览文件”查看将导入的内容。
